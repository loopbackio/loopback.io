所有端点的ACL REST API都继承了PersistedModel REST API(PersistedModel-REST-API.html)。这里提供的引用是为了方便。

默认情况下,ACL REST API不暴露。models.json可以暴露出它,添加以下:

```javascript
"acl": {
  "public": true,
  "options": {
    "base": "ACL"
  },
  "dataSource": "db"
}
```
